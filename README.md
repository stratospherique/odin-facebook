## üöÄ Getting Started

To get a local copy up and running follow these simple example steps.

### Clone

```sh
git git@github.com:stratospherique/odin-facebook.git
cd odin-facebook
```

### Install

```sh
bundle install
```

### Usage (ENV==Dev)

```sh
rails s
```
